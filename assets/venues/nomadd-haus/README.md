# Nomadd Haus Image Assets

Keep Nomadd Haus images organized by space type.

## Folder Structure

- `cover/`: main storefront or hero images
- `exhibition/`: small exhibition spaces, including 前展覽區 and ㄈㄈ房
- `video-call/`: video call fansign spaces, including ㄆㄩ房 and ㄒㄌ房
- `screening/`: small screening room images, including ㄅㄎ房
- `source-submissions/`: temporary images or links provided by the merchant before review

## Naming

Use lowercase English filenames where possible:

```text
nomadd-haus-cover-01.jpg
nomadd-haus-ff-room-01.jpg
nomadd-haus-pyu-room-01.jpg
nomadd-haus-bk-room-01.jpg
```

After images are approved, move them into the final category folder.
