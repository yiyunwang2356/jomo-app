# JOMO

JOMO is a prototype for a fan event space discovery platform.

The app helps fans and organizers search for cafes, pop-up spaces, video call rooms, album stores, and fan support venues. Stores can submit listing information, and platform admins can review listings before they become public.

## Prototype Flow

- Explore fan event venues
- Compare venue details and use cases
- Create an inquiry summary instead of checking out
- Open official store links for booking or messaging
- Let stores submit listing information
- Review submitted listings in an admin screen

## Current Build

This is a static HTML/CSS/JavaScript prototype. Open `index.html` in a browser to view it.
